# Ygoma

Yu-Gi-Oh platform for tournaments, decks, collections, trading, and blog content.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) v24+
- [pnpm](https://pnpm.io) v10+

### Install

```sh
pnpm install
```

### Development

```sh
pnpm dev
```

The web app runs at `http://localhost:3000` with the API proxied at `/api/*`.
